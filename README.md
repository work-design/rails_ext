# RailsExtend
Rails 通用基础库，对 Rails 的各个组件进行了扩展。

## 功能模块
* Ruby 核心类扩展，[链接](lib/rails_com/core)
* Rails 核心类扩展
  * ActiveStorage：[链接](lib/rails_com/active_storage) 
    * 通过 url 同步文件；
    * 将文件复制到镜像服务器；
* Rails 元信息 : Controller/Model/Routes
